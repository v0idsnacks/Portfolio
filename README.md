# Aditya Bhardwaj — Portfolio

My editorial-style developer portfolio built with React, Vite, GSAP, and CSS Variables. I focused on bold typography, momentum scrolling, and custom micro-interactions.

## Core Interactions

* **Split-Scroll Showcase**: A ScrollTrigger interaction where my left-hand column (numbers) counter-scrolls downwards while my right-hand details scroll upwards.
* **Elastic Morphing Cursor**: My custom elastic cursor follower that hides its center dot and expands into an action arrow (`↗`) on interactive elements.
* **Theme-Adaptive Headers**: Custom `theme-color` meta tags that dynamically color mobile status bars and address bars depending on light/dark mode systems.
* **Smooth Momentum**: Lenis smooth scrolling integrated natively across all viewport elements.

## Tech Stack

* **Framework**: React 18 + Vite (Bundler)
* **Animation**: GSAP 3 + ScrollTrigger
* **Smooth Scroll**: Lenis
* **Styling**: CSS Variables & Tailwind

## Quick Start

Run the project locally:

```bash
# Install dependencies
bun install

# Start development server
bun run dev

# Build for production
bun run build
```
